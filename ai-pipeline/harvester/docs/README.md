# Документация Harvester

Спецификации, дизайн и руководства по компоненту Harvester (AI-пайплайн сбора и обогащения данных).

## Структура

- **Корень этой папки** — спеки, дизайн, чек-листы по Harvester (имена без даты, например `crawler_spec.md`, `prompts_design.md`).
- **reports/** — отчёты по прогонам, аудитам, тестам. Именование: `YYYY-MM-DD__краткое-имя.md`. В начале отчёта: дата, commit, область, источник истины (например `docs/Navigator_Core_Model_and_API.md` или `docs/AI_Pipeline_Navigator_Plan.md`). Подробнее: [reports/README.md](reports/README.md).
