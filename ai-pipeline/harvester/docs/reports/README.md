# Отчёты Harvester

Здесь хранятся **отчёты** по прогонам, тестам и аудитам компонента Harvester (краулер, парсинг, интеграция с ядром).

## Именование

- `YYYY-MM-DD__краткое-описание.md` (латиница, дефисы, два подчёркивания между датой и темой).
- Примеры: `2026-02-22__crawler-run-vologda.md`, `2026-02-22__seeders-sync-report.md`.

## Метаданные в начале отчёта

Рекомендуется указывать:

- **Дата** отчёта
- **Git commit** (короткий хеш) — снимок кода, по которому сделан отчёт
- **Область** — какие модули/папки затрагивались
- **Источник истины** — например `docs/Navigator_Core_Model_and_API.md` или `docs/AI_Pipeline_Navigator_Plan.md`

Общие правила: [docs/DOCUMENTATION_RULES.md](../../../../docs/DOCUMENTATION_RULES.md).
